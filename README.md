# OC DocGen Skill



面向 OC 角色的自动化角色文档生成工具与配套数据集

------

## 项目简介



OC DocGen Skill 是一个开源项目，提供一套可直接使用的 OC 角色文档生成skill，以及用于训练、微调或参考的 标注数据集。

旨在标准化、自动化生成完整、结构清晰的 OC 人设文档，适用于创作者、AI 角色生成、对话系统、世界观搭建等场景。

## 功能特性



- 开箱即用的 OC 角色文档生成能力
- 结构化输出：基本信息、性格、背景、关系、能力、外貌等完整人设模块
- 提供高质量标注数据集，支持模型训练与微调
- 模板可自定义，方便适配不同风格与场景
- 轻量、易集成，可嵌入创作工具、AI 助手、机器人等系统

## 项目结构



```
OC-DocGen-Skill/
├── data/           # OC 角色文档标注数据集
├── skill/             # 生成技能核心逻辑与提示词模板
├── examples/           # 生成效果示例
└── README.md
```



## 使用方式



1. 克隆本项目
2. 解压，在本文件夹启用claude或codex，或把本文件夹上传给你购买的ai api（不要使用网页端免费对话）
3. 简单描述你的需求并告知加载 skill 中的模板与逻辑，实现 OC 文档自动生成
4. 根据需求修改提示词、字段结构与输出风格

## 数据集内容



数据集包含标准化 OC 人设萌属性，主要字段：

- 基础信息（姓名、性别、年龄、种族、身份）
- 外貌特征
- 性格特点
- 人际关系
- 能力与技能
- 喜好、弱点、标志性行为等
